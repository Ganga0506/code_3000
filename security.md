# Security

## Intended Users
This repository is intended for my instructor (Mat Lamoureux) and classmates in CSE 3000. It contains coursework and learning projects related to data ethics. 

## Risk Assessment
The security risk is low because the repository does not contain any sensitive personal data. However, if the code were misused, it could be modified to behave incorrectly or expose any data in the projects.

## Security Measures
No special security controls are required because this is an educational project. I have not included any private keys, passwords, or confidential info.
